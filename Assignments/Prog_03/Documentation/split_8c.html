First the program will get the argv of the input and output. Then it will create the three structs for each channel.

In order to get the filename we use basename to grab the last part of the input e.g. "image.TGA." But since we need to eliminate the .TGA file extension we copy it over to the individual channel names without the last 4 chars. While doing this we set it up so the output directory is right for each.

This double for loop will be repeated for each channel, in which it kills two channels by the position in RGBA. The code for killing the channels is from the assignment's original main.c, just modified slightly.
